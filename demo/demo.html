kanNotify

KanNotify是一个简易消息提示jquery插件。

demo
success error info warning default
success error info warning default

一·用法

$.kanNotify.add('汝之消息也');
或 
$.kanNotify.add({msg:'汝之消息也'});

二·设置

名称 说明
position 指定位置，默认为{"right":5,"bottom":50}，于屏幕右下角
type 有success、error、info、warning、default、debug六种，默认为default
msg 消息内容
icon 图标，（本作使用Font Awesome字体图标，若单独指定使用某图标，可于此直接书写对应图标之className即可），若你不想使用 Font Awesome，可在CSS中自行设置该图标样式。
wrapperWidth 设置容器宽度，默认为auto，一行而过，长短不一，若要固定宽度，侧设定wrapperWidth之值即可，默认为'auto'
allow_dismiss 默认于通知框的右上角有个关闭按钮×，可手动关闭弹出的通知框。设为false时，此按钮将不会出现。默认为true
auto_dismiss 弹出的通知框默认4000毫秒将自动关闭。设为0时，弹出的通知框将不会自动关闭。

三·方法

方法 参数 说明
add msg/options 显示一个消息通知，当参数为字符串时，该字符串为要显示的消息，并默认使用type为default的样式
success msg 显示一个消息通知，样式为success
error msg 显示一个消息通知，样式为error
info msg 显示一个消息通知，样式为info
warning msg 显示一个消息通知，样式为warning
close itemId 关闭指定id的消息通知
closeAll null 关闭所有弹出的通知
